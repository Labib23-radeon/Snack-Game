# 🐍 Classic Snake Game in Python

A modern take on the legendary **Snake Game** built using **Python** and **Pygame**. This project brings back the nostalgic arcade experience with smooth gameplay, clean visuals, and addictive mechanics.

## Project Overview

The Classic Snake Game is a complete, fully functional recreation of the timeless 90s snake game. The player controls a growing snake that eats food to increase its length and score while avoiding collisions with walls and its own body. As the snake grows longer, the speed increases, making the game progressively more challenging and exciting.

This project was developed as a fun way to strengthen my **Python programming skills** and understand core **game development concepts** such as game loops, event handling, collision detection, and real-time rendering.

## ✨ Key Features

- Smooth and responsive snake movement using arrow keys
- Dynamic difficulty — speed increases as score rises
- Real-time score tracking
- Game Over detection with final score display
- Restart functionality (Press **R**)
- Clean retro-style graphics with vibrant colors
- Simple and intuitive controls

## 🎮 How to Play

- Use **Arrow Keys** to move the snake in four directions
- Eat the red food to grow and increase your score
- Avoid hitting the walls or yourself
- Press **R** to restart after game over
- Press **Q** to quit

## 🛠️ Technologies Used

- **Python 3.x**
- **Pygame** library for graphics and game mechanics
